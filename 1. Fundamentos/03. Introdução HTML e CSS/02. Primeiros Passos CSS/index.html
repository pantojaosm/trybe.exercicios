Introdução ao CSS

Descrição

A aula de hoje fala sobre CSS, que são os estilos que
fazem parte de qualquer site na web.

Assim como o HTML, o CSS não é realmente uma linguagem
de programação. Também não é uma linguagem de marcação — é uma linguagem
de folhas de estilos. Isso significa que o CSS permite aplicar estilos
seletivamente a elementos em documentos HTML.
